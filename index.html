⏰ 시간별 예보
📅 7일 예보
📖 빵포 스케줄러 도움말
👤
① 사용자 정보 입력
계급과 성명을 입력하세요. 조원 정보도 입력하면 배너에 표시됩니다.
📅
② 월별 첫 MOR 날짜 등록
달 탭에서 해당 달 선택 → 첫 MOR 날짜 입력 → 이 달 저장.
✓ 초록 = 등록 완료 | 예상 = 미등록
🚀
③ 스케줄러 시작
정보와 날짜를 하나 이상 등록하면 시작 버튼이 활성화됩니다. 다음부터 앱 열면 캘린더 바로 표시.
💾
데이터 저장
모든 데이터는 이 기기에만 저장됩니다. 앱을 닫아도 유지됩니다.
🔄
5일 근무 사이클
아래 패턴이 반복됩니다.
일차근무시간비고
1일차MOR+MID◀07:30~12:00
00:00~07:30오전근무+자정출근
2일차MID+SWI▶~07:30퇴근
17:30~00:00새벽퇴근+저녁출근
3일차AFT12:00~17:30오후근무
4~5일차OFF비번2일 휴식
📅
월별 캘린더
MID ◀ = 자정 출근 | SWI ▶ = 자정 퇴근
예상 배지 = 미등록 달
🔴
날짜 칸 표시
● 초록점 = 메모 있음 | ● 보라점 = 조원 변경됨
📋
주별 뷰
‹ › 버튼으로 주 이동, 오늘로 버튼으로 현재 주로 이동.
⏰
타임라인 바
00~24시 타임라인에 근무 시간대가 색상 바로 표시. 오늘은 빨간 현재시각 선 표시.
✏️
날짜 탭 → 편집
🔄 변경 — 근무를 다른 것으로 교체
➕ 추가 — 기존 근무에 추가 (예: MOR + AFT)
↩ 기본 패턴으로 되돌리기 가능. 저장 버튼으로 적용.
👥
날짜별 조원 변경
특정 날만 다른 조원과 근무 시 입력. 보라색 점·카드 테두리로 표시됩니다.
💤
수면 패턴 추천
AASM·WHO·CDC 기준으로 근무 유형별 최적 취침·기상 시간 안내.
📝
메모 전체 보기
상단 탭 📝 메모에서 입력한 모든 메모를 최신순으로 확인. 항목 탭하면 해당 달 이동.
⏳
다음 상번 카운터
MOR/MID/SWI/AFT 끝난 후 다음 상번까지 남은 시간을 상시 표시합니다.
😄
시간 환산 재미 카드
다음 상번까지 남은 시간을 라면·축구·영화 등으로 환산해서 보여줍니다.
⏱️
누적 근무 통계
이달 누적 = 이번 달 오늘까지 | 전체 누적 = 모든 등록 달 합계
🏢
주간 근무자 비교
주 50시간 기준 평일(공휴일 제외) 근무자와 이달/전체 근무시간을 비교합니다.
