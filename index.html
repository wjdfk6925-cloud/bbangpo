📖 빵포 스케줄러 도움말
👤
① 조원 선택
2인 1조 편성에서 나의 이름을 탭하세요.
같은 조의 파트너가 자동으로 표시됩니다.
다른 조원과 일시적으로 교대할 경우 ⚡ 일시적 변경에서 선택하세요.
📅
② 첫 MOR 근무 날짜 등록
달 탭(1월~)에서 해당 달을 선택 후
그 달의 첫 번째 MOR 근무 날짜를 입력하고 이 달 저장을 누르세요.
달마다 시작일이 다를 수 있으므로 각각 등록하세요.
✓ 초록색 달 = 등록 완료  |  예상 = 미등록 달
🚀
③ 스케줄러 시작
조원과 날짜를 하나 이상 등록하면
📅 스케줄러 시작 버튼이 활성화됩니다.
다음부터는 앱을 열면 캘린더가 바로 표시됩니다.
💾
데이터 저장
모든 데이터는 이 기기에만 저장됩니다.
앱을 닫았다 열어도 그대로 유지됩니다.
크롬에서 인터넷 사용 기록 삭제 → 사이트 데이터 삭제 시에만 초기화됩니다.
🔄
5일 근무 사이클
아래 패턴이 반복됩니다.
일차근무시간비고
1일차 MOR + MID◀ 07:30~12:00
00:00~07:30 오전근무 후
자정에 재출근
2일차 MID + SWI▶ ~07:30 퇴근
17:30~00:00 새벽 퇴근 후
저녁 재출근
3일차 AFT 12:00~17:30 오후근무
4~5일차 OFF 비번 2일 휴식
⏱
근무 시간
■ MOR: 4시간 30분   ■ AFT: 5시간 30분
■ SWI: 6시간 30분   ■ MID: 7시간 30분
📅
월별 캘린더
각 날짜 칸에 근무명과 시간대가 표시됩니다.
◀ MID = 자정(00:00)에 출근하는 날
SWI ▶ = 자정(00:00)까지 근무하는 날
예상 배지 = 해당 달이 미등록 상태
🔢
상단 통계 바
해당 달의 MOR/AFT/SWI/MID/OFF 횟수가 표시됩니다.
각 근무 종류별 이달 일수를 한눈에 확인할 수 있습니다.
🏖
공휴일 표시
2025~2027년 대한민국 공휴일이 자동 표시됩니다.
빨간 글씨 = 일요일 또는 공휴일
파란 글씨 = 토요일
📋
주별 뷰
상단 탭에서 📋 주별을 선택하면 주간 뷰가 표시됩니다.
‹ › 버튼으로 이전/다음 주 이동
오늘로 버튼으로 현재 주로 즉시 이동
⏰
24시간 타임라인 바
각 날짜 카드에 00~24시 타임라인이 표시됩니다.
근무 시간대가 색상 바로 시각화됩니다.
빨간 세로선 = 오늘 현재 시각
◀ MID = 자정부터 근무 시작
SWI ▶ = 자정까지 근무 지속
📊
주간 요약
이번 주 근무 일수, 총 근무시간, OFF 일수, OFF 시간이 표시됩니다.
✏️
날짜 탭 → 근무 수동 변경
캘린더/주별뷰에서 날짜를 탭하면 편집 창이 열립니다.
🔄 근무 변경에서 원하는 근무를 선택 후 저장하면
해당 날짜만 개별적으로 변경됩니다.
패턴 기본값과 동일하게 선택하면 변경이 취소됩니다.
🏷️
표시 (플래그)
날짜에 중요도 표시를 달 수 있습니다.
🎯 중요   ⚠️ 주의   🎉 이벤트   💊 의무   📋 행정
캘린더 날짜 칸 우측 하단에 이모지가 표시됩니다.
📝
메모
날짜별로 자유롭게 메모를 입력할 수 있습니다.
메모가 있는 날짜 우측 상단에 ● 초록 점이 표시됩니다.
주별 뷰에서는 카드 안에 메모 내용이 바로 표시됩니다.
⚡
일시적 조원 변경
설정 화면 하단 ⚡ 일시적 변경에서 다른 조원을 선택하면
해당 조원의 스케줄로 임시 전환됩니다.
헤더에 임시변경 표시가 나타납니다.
⏱️
누적 근무 통계
캘린더 상단 통계 카드에서 확인할 수 있습니다.
이달 누적 = 이번 달 오늘까지의 합계
전체 누적 보기 = 등록된 모든 달의 합계
💪
총 근무 시간 계산
근무 시간은 X시간 Y분 / X일 X시간 X분 X초로 표시됩니다.
근무:OFF 비율이 프로그레스 바로 시각화됩니다.
🗑️
전체 초기화
헤더의 🗑️ 버튼 또는 설정 하단 전체 초기화로
모든 설정·스케줄·메모가 삭제됩니다.
⚠️ 되돌릴 수 없으니 주의하세요!
